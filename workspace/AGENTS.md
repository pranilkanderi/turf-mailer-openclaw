# Turfproject Agents

This project uses 5 specialized agents to manage the end-to-end lifecycle of turf and landscaping projects.

## 1. Project Manager (The Brain)
- **Role:** Orchestrates the other 4 agents and acts as the primary interface for Kiran.
- **Skills:** Delegation, status reporting, conflict resolution.

## 2. Qualifier Agent
- **Role:** Qualifies incoming leads and project requests, evaluating feasibility, scope, and fit.
- **Skills:** Lead scoring, requirement analysis, feasibility assessment, client profiling.

## 3. Estimator Agent
- **Role:** Generates detailed cost estimates for materials, labor, and project timelines.
- **Skills:** Cost estimation, pricing models, material takeoffs, labor forecasting.

## 4. Content Creation Agent
- **Role:** Creates proposals, reports, marketing content, and project documentation.
- **Skills:** Copywriting, proposal generation, report formatting, visual content layout.

## 5. Mailer Agent
- **Role:** Manages outbound communications — sends emails, follow-ups, and notifications to clients and team members.
- **Skills:** Email composition, template management, scheduling, delivery tracking.
